Bienvenue sur Astrolog32!

 Astrolog32 est un logiciel d'astrologie gratuit, conçu pour Windows. Ses fonctionnalités, riches et variées, en font un outil de calculs idéal pour les débutants, mais aussi pour les utilisateurs plus expérimentés qui y trouveront des fonctions techniques plus avancées.

Astrolog a été créé par Walter Pullen, qui a interrompu son développement en Décembre 1998. Valentin Abramov a continué à faire évoluer le logiciel jusqu'en Avril 2002. Astrolog32 est basé sur la version "Astrolog 5.41g pour MS Windows", modifiée par Valentin Abramov. Pour obtenir la liste des améliorations à partir de la version 5.41g, veuillez regarder dans le fichier "History A32.html".



Première utilisation d'Astrolog32

 Conseils aux nouveaux utilisateurs :

  1. Dézippez tous les fichiers dans le répertoire c:\astrolog32, en conservant les noms de répertoires définis dans le zip. Deux sous-répertoires seront créés, "principal" et "interprétations". Vous pouvez ranger Astrolog32 dans un répertoire différent, mais vous devrez alors signaler à Astrolog32 où vous l'avez placé.
  2. Double-cliquez sur "astrolog32.exe". Une fenêtre apparaîtra pour configurer Astrolog32. Vous commencerez par choisir le réglage régional le plus approprié à vos souhaits. Ensuite, entrerez les coordonnées et le fuseau horaire où vous vivez. Enfin, si vous avez installé Astrolog32 dans un répertoire différent, vous pourrez spécifier où vous l'avez placé.
  3. Nous vous recommandons de télécharger les fichiers d'Ephémérides Suisses à partir du site d'Astrolog32, ou à partir d'AstroDienst. Avec ces fichiers, non seulement vous obtiendrez des résultats plus précis, mais des calculs plus rapides.


Systèmes d'exploitation supportés

 Testé et validé sur : Windows XP, Windows 2000, Windows NT 4.0, Windows Millennium Edition, Windows 98.

Non testé, mais devant fonctionner avec : MS Windows Server 2003.

Windows 95 : La majorité des fonctions devrait bien fonctionner, mais d'autres non.  Ce logiciel n'a pas été testé sur Windows 95 et n'est pas recommandé pour cet environnement.

Ne fonctionne pas avec : MS DOS, MS Windows 3.1 ou plus ancien, MAC OS, OS/2, Unix/Linux - pour tous ces systèmes d'exploitation, il existe d'autres versions d'Astrolog qui fonctionnent très bien.


Support

 Il y a plusieurs choix pour l'assistance:

  • Vous pouvez lire le fichier "mode_emploi_astrolog32.pdf" en français.
  • Il y a aussi deux anciens fichiers en anglais, "helpfile.doc"et "helpfile.txt", très utiles mais non mis à jour.
  • Il y a le menu "Aide" dans le logiciel Astrolog32 lui-même.
  • Des aides réalisées par d'autres usagers d'Astrolog sont accessibles à partir du site Philipeau . Vous pourrez rejoindre le forum pour y poster un message.
