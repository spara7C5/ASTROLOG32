- KEYBOARD SWITCHES - the keys to the highway!

.. the case for moving on quickly makes using the keyboard switches a most natural way of working, as you gain confidence.

~ Most important ones are listed in the first batch below ~

Some switches are toggles, so the things they do can be reversed by using the same switch again. This is a relief - with Astrolog32 you can find yourself very soon in un-signposted territory.

   ALERT! the Esc key cancels pop-up windows as you would expect (but BEWARE: another Esc closes Astrolog!!).

   single keypresses drive the older and most vital switches, and these are almost all toggles.

   As many actions are invoked by combining keys. Examples are: the shiftkey switch '?' for a list of keyboard switches, 'Alt+z', to put in data for a new chart, 'Alt+w' to save the main chart data to a .dat file.... these are almost never toggles; but there are exceptions.
