Astrolog32 v. 1.20 Readme



Welcome to Astrolog32!

 Astrolog32 is a free Windows software with a rich feature set. It has most of the features used by beginners, and some used by advanced students of Astrology.

Astrolog was created by Walter Pullen, who finished development in December 1998. Valentin Abramov continued its development until April 2002.  Astrolog32 is based on Astrolog version 5.41g for MS Windows, as modified by Valentin. For a list of changes done since 5.41g, please look at the bottom of file "History A32.html".



Using Astrolog32 for the first time

 New users should do the following:
  1. Unzip all files to c:\astrolog32, keeping the same folder names as in the zip file. This will result in two sub-directories being created, "main" and "interpretations". You may store Astrolog32 in a different directory, but then on the next step you will have to let Astrolog32 know where you put it.
  2. Double-click on "astrolog32.exe". You will be asked if you want to configure Astrolog32. Say yes, and proceed with configuration. First you should choose the regional settings most appropriate to you. Next enter the coordinates and time zone of where you live. Finally you can choose the directories used by Astrolog32 . If you installed it in the default directory, no change is needed at this time, otherwise configure Astrolog32 with the directory where you stored it.
  3. We recommend that you download the Swiss Ephemeris files from the Astrolog32 site, or from AstroDienst. With those files, not only will you get more precise results, but calculations are faster.

Operating Systems Supported

 Tested and it works: Windows XP, Windows 2000, Windows NT 4.0, Windows Millennium Edition, Windows 98.

Not tested, but expected to work: MS Windows Server 2003.

Windows 95: Most of the functionality will likely work, but some won't. This software was not tested and is not recommended for Windows 95.

Will not work: MS DOS, MS Windows 3.1 or earlier, MAC OS, OS/2, Unix/Linux - for all these operating systems, there are other Astrolog versions available that work well.



Support

 There are several options:
  • You can read files "helpfile.doc" or "helpfile.txt", available in the distributed package. The contents are similar, the difference is that one is a plain text file, while the other is in Microsoft Word format. Both these files are very out of date.
  • There is an Help menu in Astrolog32.
  • Help from other Astrolog users is available from Astrolog List. You need to join the group to post a message.
